# Quickstart

Get Cipher scanning your repos in under 5 minutes.

## 1. Create an Account

Go to [cipher-ai.app](https://cipher-ai.app) and sign up with Google or email.

## 2. Install the GitHub App

1. Go to **Integrations** in the sidebar
2. Click **Install GitHub App →**
3. On GitHub, select which repositories Cipher can access
4. Authorize the OAuth request — you'll be redirected back to Cipher automatically

Once connected, you'll see your repositories listed under the GitHub integration.

## 3. Run Your First Scan

Cipher scans automatically on:
- **Pull requests** — every time a PR is opened or updated
- **Schedules** — daily or weekly full-repo scans you configure

To run a scan immediately:
1. Go to **SAST** in the sidebar
2. Click **Run Scans** in the top right
3. Select the repos to scan and click **Run**
4. Results appear in the **Scan Runs** tab within 1-3 minutes

## 4. Review Findings

- **Violations** — all findings across repos, filterable by severity
- **History** — full scan history with trends
- **PR Diffs** — findings overlaid on the exact lines changed in a PR
