# Cipher Documentation

Official documentation for [Cipher](https://cipher-ai.app) — AI-powered security scanning for your codebase.

## Structure

- `getting-started/` — Installation, GitHub App setup, first scan
- `features/` — Feature-by-feature guides (SAST, Secrets, Deps & CVEs, DAST, Cloud, etc.)
- `integrations/` — Slack, Email, Jira, GitHub App
- `governance/` — Risk Scores, Scorecards, Rules, Violations
- `api/` — API reference for CLI and CI/CD usage
